# AUTO CUT

動画をアップロードするだけで、スポーツハイライト風の編集を全自動で行うブラウザ完結の動画編集ツール。サーバーには何も送信されず、すべて `ffmpeg.wasm` によりブラウザ内で処理されます。

## できること

1. 動画内の「動き」（フレーム差分）と「音の盛り上がり」（音量エネルギー）を解析し、見せ場となるカットを自動選定
2. 選定したカットをつなぎ合わせ、コントラスト/彩度アップ・ビネットで映像を引き締め
3. 一番スコアの高いカットをスローモーションにして最後に配置（決めのシーン演出）
4. 各カットにキャプションを自動表示、全体にフェードイン/アウトを付与
5. mp4（H.264/AAC）として書き出し、ダウンロード可能

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 本番ビルド
```

`npm install` 時に `postinstall` で `@ffmpeg/core`（シングルスレッド版）を `public/ffmpeg/` へコピーし、外部CDNなしで自前ホストします（`public/ffmpeg/` はgit管理外）。`public/Anton.ttf` はキャプション用フォント（Anton, OFL-1.1）。
